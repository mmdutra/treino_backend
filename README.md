# Teste de backend

API criada com a finalidade de estudar um pouco mais sobre o micro-framework Lumen. 

## Ferramentas utilizadas

* Framework [Lumen 7.0](https://lumen.laravel.com)
* PHP 7.3

- **Configuração**
    + ``` composer install ```
    + ``` cp .env.example .env ```
    + ``` php artisan key:generate ```
    + ``` php artisan migrate ```
